# 📘 Página Web: Aplicación de Inteligencia Artificial en el Aula

## 📌 Descripción

Este proyecto consiste en el desarrollo de una página web utilizando **HTML5 y CSS3**, con el objetivo de presentar de forma clara y estructurada el uso de la **Inteligencia Artificial (IA) en el ámbito educativo**.

La página incluye contenido informativo, navegación estructurada, un formulario de contacto funcional y estilos visuales básicos aplicados mediante CSS externo.

---

## 🎯 Objetivos

* Aplicar la estructura semántica de HTML5.
* Organizar correctamente los elementos de contenido.
* Implementar un formulario funcional.
* Utilizar estilos CSS3 en un archivo externo.
* Desarrollar una página web temática sobre tecnología.

---

## 🧠 Temática

**Aplicación de la Inteligencia Artificial en el Aula**

La IA permite mejorar el proceso educativo mediante herramientas que facilitan el aprendizaje personalizado, la automatización de tareas y el uso de asistentes virtuales.

---

## 🧱 Estructura del Proyecto

El proyecto está organizado de la siguiente manera:

```
📁 proyecto

```

---

## 🧩 Estructura HTML5 utilizada

La página implementa etiquetas semánticas como:

* `<header>` → Encabezado principal
* `<nav>` → Menú de navegación
* `<main>` → Contenido principal
* `<section>` → Secciones informativas
* `<footer>` → Pie de página

---

## 🎨 Estilos CSS

Los estilos fueron desarrollados en un archivo externo (`styles.css`) incluyendo:

* Diseño básico y limpio
* Uso de colores para mejorar la visualización
* Estilización del formulario
* Estructura adaptable

---

## 📬 Formulario de Contacto

Se incluye un formulario funcional con los siguientes campos:

* Nombre
* Correo electrónico
* Mensaje

Con validaciones básicas utilizando el atributo `required`.

---

## 🚀 Cómo ejecutar el proyecto

### Opción 1: Navegador

1. Abrir el archivo `index.html` con doble clic.

### Opción 2: Visual Studio Code (recomendado)

1. Abrir el proyecto en Visual Studio Code.
2. Instalar la extensión **Live Server**.
3. Ejecutar con clic derecho → *Open with Live Server*.
